База JSON → Leaflet/OSM
Загрузи базу из JSON (файл или URL). Все дома рисуются сразу. Если контур неверный — включи «Режим привязки», кликни на карте, выбери найденный контур и сохрани в базу.
Загрузка базы
Можно открыть со строкой запроса: ?data=<raw_json_url> — база подгрузится автоматически.

Привязка/коррекция полигона
1) Выбери запись в списке → 2) Нажми «Режим привязки» → 3) Кликни по карте рядом с домом → 4) Подтверди сохранение.
